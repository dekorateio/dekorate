What is Dekorate?

Dekorate is a collection of Java compile-time generators and decorators for Kubernetes/OpenShift manifests. It makes generating Kubernetes manifests as easy as adding a dependency to the classpath of your project and customizing as simple as setting an annotation or application property. Stop wasting time editing xml, json and yml and customize the kubernetes manifests as you configure your java application.
